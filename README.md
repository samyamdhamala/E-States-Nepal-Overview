# 🏠 E-States Nepal

A full-stack property listing platform built using Flutter (Frontend) and Express.js + MySQL (Backend).

---

## 🔧 Technologies Used
- **Frontend**: Flutter
- **Backend**: Node.js (Express), MySQL
- **Other**: JWT, REST API, SDLC Documentation
